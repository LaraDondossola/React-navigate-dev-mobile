## 🍳 Atividade: App de Receitas (Navegação com Bottom Tabs)

Este projeto em específico é o resultado de uma atividade prática solicitada durante a aula. O objetivo principal foi desenvolver um aplicativo funcional utilizando **React Native** com **Expo**.

O desafio designado ao nosso grupo foi a implementação do **Native Bottom Tabs**. Desenvolvemos um exercício simples focado na criação de telas organizadas por abas inferiores, onde o tema escolhido para o aplicativo foi um catálogo de receitas.

### 🛠️ Funcionalidades
* Navegação intuitiva entre diferentes telas através de abas inferiores.
* Interface simples e funcional para visualização de pratos.
